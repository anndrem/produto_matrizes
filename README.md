# Multiplicação de Matrizes em Python

Uma implementação simples e eficiente de multiplicação de matrizes utilizando Python puro.

---

## 📌 Sobre o Projeto

Este repositório contém uma implementação em Python de multiplicação de matrizes **sem o uso de bibliotecas externas como NumPy**.

O objetivo do projeto é reforçar a compreensão de:

- Laços aninhados (nested loops)
- Representação de matrizes com listas
- Validação de dimensões
- Complexidade algorítmica
- Fundamentos de Álgebra Linear

---
